# Вариант 11 — Программа шифрования

## Требования задания

- **Шифр Плейфера с 4 таблицами** — английский текст
- **Шифр Виженера (прямой ключ)** — русский текст  
- Java-приложение с графическим интерфейсом (не консоль)
- Загрузка/сохранение файлов
- Ввод/вывод текста с клавиатуры
- Игнорирование символов не из заданного алфавита
- Ключ задаётся пользователем

## Сборка и запуск

```bash
./compile.sh   # компиляция
./run.sh       # запуск
```

## Структура проекта

- `src/PlayfairCipher.java` — шифр Плейфера с 4 матрицами (английский, I=J)
- `src/VigenereCipher.java` — шифр Виженера с прямым ключом (русский, включая Ё)
- `src/CipherApp.java` — главное окно и логика

## Использование

1. Выберите алгоритм в выпадающем списке.
2. Введите 4 ключа (Плейфер) или 1 ключ (Виженер).
3. Введите или загрузите текст.
4. Нажмите «Зашифровать» или «Расшифровать».
5. Сохраните результат в файл при необходимости.

## Тестирование (по заданию)

### Плейфера
- Дымовое: фраза с недопустимыми символами, 4 ключа, таблицы, биграммы.
- Валидные крайности: чётная/нечётная длина текста, подряд одинаковые буквы (вставка X).
- Невалидные: коллизии XX (вставка Z между ними).

### Виженера
- Дымовое: фраза с недопустимыми символами, ключ, таблица подстановки.
- Валидные: текст с буквой Ё.
- Невалидные: ключ с недопустимыми символами.
